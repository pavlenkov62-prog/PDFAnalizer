# PDFAnalizer

Проект анализа структуры PDF-документов.

## Цель

Построить логическую модель документа, независимую от внутреннего устройства PDF.

## Основные этапы

1. Анализ PDF
2. Выделение объектов страницы
3. Построение модели документа
4. Перевод текстовых объектов
5. Новая компоновка
6. Генерация нового PDF
